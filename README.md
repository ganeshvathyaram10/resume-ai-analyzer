# 🤖 Resume AI Analyzer

An AI-powered web app that analyzes your resume against a job description and gives instant feedback.

🔗 **Live Demo:** [resumeaianalyzer-vathy.netlify.app](https://resumeaianalyzer-vathy.netlify.app)

---

## ✨ Features

- 📊 Match score out of 100
- ✅ Strengths in your resume
- ❌ Missing skills for the job
- ✏️ AI-improved bullet points
- 💡 One line overall summary

---

## 🛠️ Built With

- Python, FastAPI, Groq API (LLaMA 3.3 70B)
- HTML, CSS, JavaScript
- Deployed on Railway + Netlify
